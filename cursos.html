Cursos
de Tecnologia!

Descubra o poder de transformar ideias em realidade com
apenas algumas linhas de código!

Front-End

No Front-End, a pessoa cria a parte visual de sites com HTML, CSS e JavaScript.

HTML Básico

HTML é a base da web, usada para estruturar e organizar o conteúdo de uma página, como textos, imagens e links.

CSS Básico

CSS é usado para estilizar páginas web, definindo cores, fontes, layouts e o design visual do site.

JavaScript Básico

JavaScript torna as páginas web interativas, permitindo animações e respostas rápidas às ações do usuário.

Back-End

No Back-End, a pessoa aprende a manipular dados, criar APIs e conectar o sistema ao banco de dados.

Arduino

No Arduino, a pessoa aprende a programar e controlar sensores e motores para criar projetos interativos.

Design

No Design, a pessoa aprende a criar layouts visuais com cores e tipografia.

Kotlin

No Kotlin, a pessoa aprende a programar aplicativos Android de forma simples e segura.
